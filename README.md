# Project Title

E-commerce Project

## Description

This project is an E-commerce platform designed to provide users with a convenient and seamless shopping experience. 
## Technologies Used

    Frontend:
        HTML, CSS, JavaScript
        React.js (frontend library)
        Bootstrap (for basic CSS)
    Backend:
        Node.js
        Express.js
        MYSql (database)
    Authentication:
        JSON Web Tokens (JWT)
    Other tools:
        Git (for version control)
